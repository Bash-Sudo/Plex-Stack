Plex Claim Token

Two different Plex tokens — here is the difference:

🔑 Claim Token (this step) — a one-time code from plex.tv/claim used to link your new Plex server to your Plex account on first launch. Expires in 4 minutes.

🔐 Plex Token (added later in Settings) — a permanent token for API access used by Prefetcharr. You get this after Plex is running, from Settings in the control panel.
Get this right before clicking Deploy Plex — it expires in 4 minutes
  1. Click plex.tv/claim — sign in to your Plex account if prompted
  2. Copy the claim-xxxxxxxx token shown on that page
  3. Paste it below, then immediately click Deploy Plex
If the timer expires, just go back to plex.tv/claim and paste a fresh one — takes 5 seconds.
